To use in Shopify to display recently viewed products.

h2. Dependencies

jQuery > 1.3
jQuery template plugin ("http://api.jquery.com/category/plugins/templates":http://api.jquery.com/category/plugins/templates)

h2. Installation

http://wiki.shopify.com/Recently_Viewed_Products

h2. Configuration

Configuration parameters are all optional.

You pass an object to Shopify.Products.showRecentlyViewed().

* **howManyToShow** - how many recently viewed products to show / list. Default value: 3. A number. Optional.
* **howManyToStoreInMemory** - how many recently viewed products to store in memory (the handle of the product is stored in a cookie that lives for 3 months). Default value: 10. A number. Optional.
* **wrapperId** - the id of the element in which to render the recently viewed product temmplate. If none is specified, the script will look for a div or ul (or whatever element) with id 'recently-viewed-products' in the document. A string. Optional. Careful: don't use '#', just put here the value of the id attribute.
* **templateId** - the id of the 'text/x-jquery-tmpl' template to use to render a recently viewed product. That's your presentational layer. The HTML generated will be appended to the insides of the 'wrapperId' element. Optional. If no template is specified, the script will look for one with id 'recently-viewed-product-template'. A string. Careful once more: don't use '#'â€š just put here the value of the id attribute. What's a template? See here: "http://api.jquery.com/category/plugins/templates":http://api.jquery.com/category/plugins/templates.
* **onComplete** - A function to run when all recently viewed products have been rendered on the page. Totally optional. Use this if you need to do anything special when all is said and done. (You need a callback function here because Ajax calls are used to fetch information about the recently viewed products, to show a thumbnail, their price, etc.)
